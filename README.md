# 🐍 100 Dias of Code — Python

> Repositório de atividades e projetos desenvolvidos ao longo do desafio **100 Days of Code** com foco em Python.

---

## 📌 Sobre o repositório

Este repositório reúne os exercícios, mini-projetos e desafios práticos resolvidos dia a dia durante o curso **100 Days of Code: The Complete Python Pro Bootcamp**. O objetivo é documentar a evolução do aprendizado e manter um histórico organizado das atividades.

---

## 🗂️ Estrutura

```
100-Days-Python/
│
├── Dia 1/       # Variáveis, tipos de dados e manipulação de strings
├── Dia 2/       # Compreensão de dados e operações matemáticas
├── Dia 3/       # Estruturas condicionais
├── Dia 4/       # Listas e funções aleatórias
├── Dia 5/       # Loops e controle de fluxo
├── Dia 6/       # Funções e blocos de código
├── Dia 7/       # Lógica e jogos simples (ex: Hangman)
├── ...
└── Dia-100/       # Projeto final
```

> Cada pasta contém o(s) arquivo(s) `.py` referente(s) ao dia, além de um `README.md` opcional com a descrição do exercício.

## ▶️ Como executar

1. Clone este repositório:
   ```bash
   git clone [https://github.com/seu-usuario/100-Dias-of-code-python.git](https://github.com/KakaPerpetuo/100-Days-Pyhton.git)
   ```

2. Acesse a pasta do dia desejado:
   ```bash
   cd Dia 1
   ```

3. Execute o script Python:
   ```bash
   python main.py
   ```

> Certifique-se de ter o **Python 3.x** instalado. Alguns dias podem exigir dependências extras listadas em um `requirements.txt` local.
